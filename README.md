#  Sales Insights Data Analysis Project

## Project Overview
This project analyzes sales transaction data to uncover revenue trends, market performance, and customer contribution patterns. The objective was to generate actionable business insights to support strategic decision-making.

##  Business Objectives
The company aimed to:
- Identify high-revenue markets  
- Analyze monthly and yearly revenue trends  
- Detect seasonal sales patterns  
- Determine top revenue-generating customers  
- Understand regional revenue contribution  
- Handle currency inconsistencies in transaction data  

## Tools & Technologies Used
- MySQL  
- SQL  
- Power BI  
- ETL (Data Cleaning & Transformation)  

## Data Cleaning & Preparation
- Imported raw transaction data into MySQL  
- Validated and cleaned inconsistent records  
- Converted USD transactions to INR for consistency  
- Joined transaction and date tables for time-based analysis  

## Key KPIs & Metrics
- Total Revenue (Year-wise & Market-wise)  
- Monthly Revenue Trends  
- Revenue by Market  
- Revenue by Customer  
- Market Contribution Perce
